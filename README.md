# Habit Tracker 📅

A Python-based Habit Tracker application that helps users create, manage, and track daily habits with streak tracking.

## ✨ Features

- Add new habits
- View existing habits
- Mark habits as completed
- Track daily progress
- Maintain habit streaks
- Store data using JSON file handling

## 🛠️ Technologies Used

- Python
- JSON (Data Storage)
